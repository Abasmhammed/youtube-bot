## 🌟 الخطوة 1: الحصول على مفتاح البث من يوتيوب

### طريقة الحصول على YouTube Stream Key:

1. **اذهب إلى YouTube Studio:**
   - افتح [studio.youtube.com](https://studio.youtube.com)
   - تأكد من تسجيل الدخول بحسابك

2. **انقر على "Go Live" أو "البث المباشر":**
   - من القائمة الجانبية، اختر **Create** أو **الإنشاء**
   - اختر **Go Live** أو **البث المباشر**

3. **اختر "Stream Setup" أو "إعداد البث":**
   - اختر **Custom Ingest** (البث المخصص)

4. **انسخ مفتاح البث:**
   ```
   Server URL (الخادم): rtmp://a.rtmp.youtube.com/live2/
   Stream Key (مفتاح البث): xxxx-xxxx-xxxx-xxxx
   ```

   ⚠️ **تحذير مهم:** لا تشارك مفتاح البث مع أحد!

## 📹 الخطوة 3: إضافة الفيديوهات

### خيارات لإضافة الفيديوهات:

#### الطريقة 1: عبر الويب (GitHub Web UI)
1. افتح مستودعك على GitHub
2. افتح مجلد `videos`
3. انقر على **Add file** → **Upload files**
4. اختر ملفات الفيديو (MP4, MKV, MOV)
5. انقر **Commit changes**

#### الطريقة 2: عبر سطر الأوامر (Git CLI)
```bash
# نسخ الفيديوهات إلى مجلد videos
cp /path/to/your/videos/*.mp4 videos/

# إضافة الملفات إلى Git
git add videos/

# حفظ التغييرات
git commit -m "add: new motivational videos"

# رفع إلى GitHub
git push
```

#### الطريقة 3: استخدام GitHub Desktop
1. افتح GitHub Desktop
2. اختر المستودع
3. اسحب الملفات إلى مجلد `videos`
4. أنشئ Commit وادفع التغييرات

### ✅ الملفات المدعومة:
- `.mp4` - الأكثر شيوعاً والأفضل
- `.mkv` - جودة عالية
- `.mov` - من أجهزة Apple

## 🔐 الخطوة 4: إضافة مفتاح البث كـ Secret

### عبر موقع GitHub:

1. **افتح إعدادات المستودع:**
   - انقر على **Settings** في مستودعك

2. **اختر Security → Secrets and variables → Actions:**
   - انقر على **New repository secret**

3. **أضف المفتاح:**
   - **Name:** `YOUTUBE_STREAM_KEY`
   - **Value:** (ألصق مفتاح البث الذي نسخته)
   - انقر **Add secret**

✅ الآن مفتاح البث آمن وغير مرئي لأي شخص!

## ⚙️ خيارات إضافية

### تغيير جدولة البث:

عدّل الملف `.github/workflows/stream.yml`:

```yaml
on:
  schedule:
    - cron: '0 */6 * * *'  # كل 6 ساعات
```

#### أمثلة الجدولة:
```
'0 * * * *'      → كل ساعة
'0 */2 * * *'    → كل ساعتين
'0 */3 * * *'    → كل 3 ساعات
'0 */6 * * *'    → كل 6 ساعات (الافتراضي)
'0 12 * * *'     → يومياً في الظهيرة
'0 0 * * 0'      → أسبوعياً يوم الأحد
```

### تخصيص جودة البث:

عدّل `env` في `.github/workflows/stream.yml`:

```yaml
env:
  VIDEO_BITRATE: '3000k'     # 3000k = جودة عالية, 1000k = أقل
  AUDIO_BITRATE: '128k'      # 128k = جود عالية, 64k = أقل
  MAX_RETRIES: '3'           # عدد محاولات إعادة الاتصال
```

## 🆘 استكشاف الأخطاء الشائعة

### ❌ "YOUTUBE_STREAM_KEY not found"
**الحل:** تأكد من إضافة المفتاح في GitHub Secrets بالاسم الصحيح

### ❌ "No videos found"
**الحل:** أضف فيديوهات في مجلد `videos/` وادفعها إلى GitHub

### ❌ "ffmpeg: command not found"
**الحل:** FFmpeg سيتم تثبيته تلقائياً (تحقق من السجلات)

### ❌ "Connection refused"
**الحل:** تحقق من الإنترنت واتصالك بـ YouTube

### ❌ البث يعمل لكن لا أرى الفيديوهات
**الحل:** 
1. تحقق من أن الفيديوهات موجودة في YouTube Studio
2. اعد تحميل الصفحة
3. جرب تشغيل البث مرة أخرى
