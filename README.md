# 🎬 YouTube Live Bot - بوت البث المباشر على يوتيوب

بوت متقدم وذكي يقوم ببث الفيديوهات التحفيزية بشكل مستمر 24/7 على قناتك في يوتيوب، مما يساعدك على جذب مشاهدات أكثر وبناء جمهور متفاعل.

## ✨ المميزات

- **بث مستمر 24/7:** البوت يعمل بشكل مستمر دون توقف
- **فيديوهات متنوعة:** يتم اختيار الفيديوهات بشكل عشوائي لضمان تنوع المحتوى
- **تلقائي تماماً:** لا تحتاج للتدخل اليدوي بعد التثبيت
- **سهل الاستخدام:** خطوات بسيطة للبدء
- **مراقبة السجلات:** تتبع حالة البث من خلال السجلات

## 🚀 البدء السريع

### المتطلبات
- حساب GitHub
- قناة يوتيوب
- مفتاح البث (Stream Key) من يوتيوب

### خطوات التثبيت

#### 1. استنساخ المستودع
```bash
git clone https://github.com/YOUR_USERNAME/youtube-bot.git
cd youtube-bot
```

#### 2. إضافة الفيديوهات
ضع ملفات الفيديو (MP4, MKV, MOV) في مجلد `videos/`:
```bash
cp /path/to/videos/*.mp4 videos/
```

#### 3. إضافة مفتاح البث
1. اذهب إلى إعدادات المستودع (Settings)
2. اختر **Secrets and variables** → **Actions**
3. أضف متغير جديد:
   - **Name:** `YOUTUBE_STREAM_KEY`
   - **Value:** مفتاح البث الخاص بك من يوتيوب

#### 4. تفعيل GitHub Actions
1. اذهب إلى تبويب **Actions**
2. اختر **YouTube Live Stream 24/7**
3. اضغط **Run workflow**

## ⚙️ الإعدادات

### تغيير تكرار البث
عدّل السطر التالي في `.github/workflows/stream.yml`:
```yaml
- cron: '0 */6 * * *'  # غيّر الرقم 6 إلى عدد الساعات المطلوب
```

### إضافة فيديوهات جديدة
1. أضف الفيديوهات إلى مجلد `videos/`
2. ارفعها إلى GitHub
3. سيتم اختيارها تلقائياً في الدورة التالية

## 🎥 نصائح لتحسين المشاهدات

1. **عنوان جذاب:** استخدم عنوان يعكس محتوى الفيديوهات
2. **صورة مصغرة احترافية:** اختر صورة تعبر عن القوة والنجاح
3. **كلمات مفتاحية:** استخدم كلمات مثل "motivation"، "success"، "discipline"
4. **وصف مفصل:** اكتب وصفاً يشرح محتوى البث
5. **تفاعل مع المتابعين:** رد على التعليقات والأسئلة

## 🔧 استكشاف الأخطاء

### البث لا يبدأ
- تأكد من إضافة مفتاح البث في الإعدادات
- تحقق من أن الفيديوهات موجودة في مجلد `videos/`

### الفيديوهات لا تظهر
- تأكد من أن الملفات بصيغة MP4 أو MKV
- تحقق من أن حجم الملفات معقول (أقل من 500MB)

### مشاكل في الاتصال
- تحقق من سرعة الإنترنت
- جرب إعادة تشغيل البث من تبويب Actions

## 📊 مراقبة البث

2. اختر **YouTube Live Stream 24/7**
3. اضغط على آخر تشغيل لرؤية السجلات
